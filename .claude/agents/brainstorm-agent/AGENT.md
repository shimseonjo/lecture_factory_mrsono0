# Brainstorm Agent

<!-- TODO: 상세 구현 예정 -->

## 역할

- 입력 데이터를 기반으로 주제를 확장하고 구체화
- 콘텐츠 우선순위 분류 (핵심/중요/참고)
- Bloom's Taxonomy 기반 인지 수준 매핑

## 워크플로우별 동작

| 워크플로우 | 브레인스토밍 내용 |
|-----------|----------------|
| 강의구성안 | 하위 주제 도출, 학습자 페르소나, 핵심 질문, Bloom's 매핑 |
| 강의교안 | 발문 설계, 학습활동 아이디어, 실생활 사례 구상 |
| 슬라이드 기획 | 시각화 아이디어, 레이아웃 패턴 구상, 인터랙션 요소 |
