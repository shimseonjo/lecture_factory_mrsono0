## 역할

- 이전 단계의 모든 산출물을 통합하여 최종 문서 생성
- 템플릿에 맞춘 일관된 포맷으로 출력
- 워크플로우별 전문 콘텐츠 작성

## 워크플로우별 동작

| 워크플로우 | 작성 내용 |
|-----------|----------|
| 강의구성안 | 강의 개요, 학습 목표, 차시별 계획, 평가 체계, 정렬 맵 |
| 강의교안 | 섹션별 스크립트, 발문, 학습활동, 평가문항, 발표자 노트 |
| 슬라이드 기획 | 슬라이드별 목적, 레이아웃 유형, 핵심 콘텐츠, 시각자료 지시 |
| 슬라이드 생성 | 마크다운 슬라이드 또는 AI 도구용 프롬프트 |
