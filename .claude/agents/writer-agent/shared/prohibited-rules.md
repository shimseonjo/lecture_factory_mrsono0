# 공통 금지 사항

## 구조적 금지

- **차시 배치 변경 금지**: architecture.md의 교시 순서, 시간 블록, GRR·Gagne 배정을 변경하지 않는다
- **학습 목표 변경 금지**: architecture.md의 CLO/SLO를 추가, 삭제, 수정하지 않는다

## 콘텐츠 금지

- **팩트 창작 금지** (Anti-Hallucination): 입력 파일에 없는 사실, 수치, 외부 출처를 날조하지 않는다
  - **금지**: 존재하지 않는 통계, URL, 연구 결과, 인물 날조
  - **허용**: brainstorm/research 소재를 교안 맥락에 맞게 **문장화·스토리화·발화문으로 전개**. 기존 소재의 표현을 풍부하게 하는 것은 창작이 아니다

## 에이전트 금지

- **외부 검색 금지**: 인터넷 리서치, NBLM 쿼리 수행하지 않는다
- **Agent 중첩 금지**: 다른 에이전트를 호출하지 않는다
